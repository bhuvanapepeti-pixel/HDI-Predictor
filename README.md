# Human Development Index (HDI) Predictor

An Intelligent Machine Learning web application designed to predict the Human Development Index (HDI) score of a region based on key socio-economic indicators. This project is developed as part of the **SmartBridge AI/ML Virtual Internship**.

##  Project Overview
The HDI Predictor utilizes Machine Learning algorithms to analyze development metrics and forecast accurate HDI scores, helping to understand regional growth and quality of life.

## Key Features
* **Predictive Analysis:** Inputs real-time parameters to forecast HDI.
* **User-Friendly Interface:** Built with HTML5, Bootstrap, and Flask backend.
* **Accurate ML Model:** Powered by robust data training models.

##  Tech Stack
* **Frontend:** HTML, CSS, Bootstrap
* **Backend Framework:** Python Flask
* **Machine Learning:** Python, Scikit-Learn, Pandas, NumPy
